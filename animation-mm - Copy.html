မလိုပါ။ Script ရေးသားခြင်းသည် package အားလုံးတွင် ပါဝင်သည်။ သင့်အိုင်ဒီယာ၊ ဇာတ်လမ်း သို့မဟုတ် message ကိုသာ ပြောပြပါ — animation အတွက် script ကို ကျွန်ုပ်ဖန်တီးပေးမည်။ ရှိပြီးသား script ရှိပါက visual storytelling အတွက် ပြန်လည်ဖန်တီးပေးမည်။
Essential Story သည် ရိုးရှင်းသော ရှင်းလင်းချက်၊ social media content နှင့် cute loop များအတွက် သင့်တော်သည်။ Cinematic Story သည် စိတ်ခံစားမှုပြင်းထန်သော ဇာတ်လမ်းများ — မင်္ဂလာဆောင်၊ brand story၊ နက်နဲသော ကြော်ငြာများအတွက် သင့်တော်သည်။ Premium သည် အရည်အသွေးကို ဦးစားပေးသော broadcast-level ကြော်ငြာများအတွက် ဖြစ်သည်။ မသေချာပါက သင့်အိုင်ဒီယာဖြင့် ဆက်သွယ်ပါ — သင့်တော်သော package ကို အကြံပြုပေးပါမည်။
Animation ကို frame တစ်ခုချင်းစီ တည်ဆောက်ရသည်။ Scene တစ်ခု animate လုပ်ပြီးသည်နှင့် ဇာတ်ကောင်နေရာ၊ ဖွဲ့စည်းမှု သို့မဟုတ် ဇာတ်လမ်း flow ပြောင်းလဲမှုသည် frame အားလုံးကို အစကနဦး ပြန်လုပ်ရသော လုပ်ငန်းဖြစ်သည်။ ထို့ကြောင့် storyboard အတည်ပြုမှုသည် အရေးကြီးဆုံးအဆင့်ဖြစ်ပြီး — production မစတင်မီ အဓိကဆုံးဖြတ်ချက်များ အားလုံး အတည်ပြုထားရမည်။
Brief တွင် သဘောတူထားသော resolution နှင့် aspect ratio ဖြင့် MP4 (H.264) ဖော်မတ် ဖြစ်သည်။ Standard မှာ 1920×1080 (16:9) ဖြစ်သည်။ Social media အတွက် Vertical (9:16) နှင့် Square (1:1) တို့ကို add-on အဖြစ် ရယူနိုင်သည်။
ဟုတ်ကဲ့။ နောက်ဆုံးငွေပေးချေပြီးသည်နှင့် commercial rights အပြည့်အဝ လွှဲပြောင်းပေးသည်။ ကြော်ငြာ၊ social media၊ TV၊ presentation နှင့် အခြား commercial ရည်ရွယ်ချက်များအတွက် animation ကို အသုံးပြုနိုင်သည်။ မှတ်ချက်: music rights သည် ရွေးချယ်ထားသော license အမျိုးအစားပေါ် မူတည်သည်။
